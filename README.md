## Overview
Full stack responsive chat app using sockets for realtime communication. We will also be able to create groups and upload files and download files. We have used React, Node.js, Express, MongoDB, Socket.io, ReduxToolkit and many more awesome tech to create this app

## About Chat-App(Chatwave)

- ** Authentication using Node Express Mongo with JWT and Check User in Protected Routes in Middelwares
- ** User Sign in Feature with Image Support and Image Store using Cloudnary
- ** Design in Tailwind CSS, ShadCn

## Tech Stack
### Frontend
- **React.js**: Built with React ⚛️
-**UI Componenet** :  ShadCN 🧩
-**Styling** :  Tailwind CSS 🎨
- **Responsiveness**: Fully Responsive Design 📱💻
- **React Router Dom**: For navigation.
- **Axios/Fetch API**: For making HTTP requests.
- **Socket**: For Real Time Comminication.
- **Moment.js**: For making to show real time.
- **React-Icons**: For All Icons show in site.
- **Chat Support** : Group Chats and Personal Messages 👥💬
- **Fun Conversations** : Emoji Support 😄
- **State Management** : ReduxToolkit

## Tech Stack
### Backend
- **Node.js**: For server-side runtime.
- **Express.js**: For building the RESTful API.
- **Axios/Fetch API**: For making HTTP requests.
- **MongoDB/PostgreSQL**: For database management.
- **JWT**: For secure user authentication.
- **Bcryptjs**: For Password Protected in hash password.
- **Cloudinary**: For Storing Image.
- **Multer**: For for file Uploading.
- **Real-time Messaging:**: Sockets for Instant Communication 💬
